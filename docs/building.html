Building CSI

* General disclaimer:
*   The authors make no guarantee whatsoever that CSI will build on your system.
*   This is a best-effort guide with possibly-helpful comments.

Build Command

To build CSI, all you should need to do is run
scons
in the root directory of this release.

The following non-standard packages are required:

  • LLVM 3.1+ with the Clang front end
  • SCons

To run the unit test cases for CSI, run
scons test
in the root directory of this release. The following non-standard packages are required:

  • Python 2.6+

CSI is capable of aggressive optimization for some instrumentation schemes. See the detailed build comments for how to hook up with your installation of either the GAMS optimization framework or GUROBI optimizer.

Having problems? See some additional comments.
